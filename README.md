# Warga Sigap 
Warga Sigap adalah aplikasi android tanggap darurat berbasis AI yang membantu masyarakat Indonesia menghadapi bencana secara cepat, tepat, dan terarah.

## Fitur Utama
- Tombol Bantuan Darurat  
- Tombol Tanya AI
- Panduan Darurat Terstruktur  
- Info gempa real-time (BMKG)

## Teknologi
- Kotlin
- Jetpack Compose
- Retrofit
- BMKG API
- Google Play Services (Location API)

## Tujuan
Membantu masyarakat tetap tenang dan mengambil keputusan yang tepat saat menghadapi bencana.
